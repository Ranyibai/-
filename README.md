阿：(a)用在名称，小名或亲缘关系之前来表示亲昵，如“阿哥” “阿爸” “阿龙”等。(b)还用于翻译作品中的各种名称如“阿门” “阿庇乌斯” “阿拉库巴通巴通”

安：有安置，安全，安定等意思，该字常用语部分烂俗偶像剧或青春剧的男女主名称中，似乎能让作者和读者觉得这是好人。

    安全：指不会受到心灵或身体上的伤害的形容词。
